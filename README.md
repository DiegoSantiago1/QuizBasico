# 💻 Quiz Interativo para Programadores Iniciantes

Um quiz simples e interativo desenvolvido com HTML e CSS para testar conhecimentos básicos de programação.

## 🌍 Acesse o projeto online

👉 https://DiegoSantiago1.github.io/QuizBasico/

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de praticar conceitos fundamentais de desenvolvimento web, como:

- Estruturação com HTML
- Estilização com CSS
- Uso de formulários
- Elementos interativos como `<details>` e `<summary>`

O quiz contém perguntas básicas voltadas para iniciantes na área de programação.

---

## 🚀 Funcionalidades

- ✅ Perguntas de múltipla escolha  
- ✅ Campo de texto  
- ✅ Input de senha  
- ✅ Seleção de data  
- ✅ Checkboxes  
- ✅ Upload de arquivo  
- ✅ Menu suspenso (select)  
- ✅ Exibição de respostas  
- ✅ Tabela de pontuação  
- ✅ Formulário de feedback  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3
